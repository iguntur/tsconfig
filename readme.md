## Install

```console
$ npm install @guntur/tsconfig
```

## Usage

**`tsconfig.json`**

```json
{
    "extends": "@guntur/tsconfig"
}
```

| Presets                    | Target        |
|----------------------------|---------------|
| `@guntur/tsconfig`         | NodeJS `>= 6` |
| `@guntur/tsconfig/node/v4` | NodeJS `>= 4` |
| `@guntur/tsconfig/esnext`  | ES Module     |
